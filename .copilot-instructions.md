# Playstore Project - Development Instructions

## 📦 Project Setup

This is a TanStack Start project with React, TypeScript, and Tailwind CSS.

### Installation
```bash
npm install
# or
bun install
```

### Development
```bash
npm run dev
# or
bun dev
```

### Build
```bash
npm run build
# or
bun build
```

## 🔧 Environment Variables

Create a `.env.local` file with your Supabase credentials:
```
VITE_SUPABASE_URL=your_supabase_url
VITE_SUPABASE_ANON_KEY=your_supabase_anon_key
```

## 📁 Project Structure

- `/src` - Main source code
- `/public` - Static assets
- `/components` - React components
- `/pages` - Page components
- `/styles` - Global styles

## 🎨 UI Components

This project uses shadcn/ui components with Radix UI primitives and Tailwind CSS.

### Adding Components
```bash
npx shadcn-ui@latest add [component-name]
```

## 📋 Available Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run preview` - Preview production build
- `npm run lint` - Run ESLint
- `npm run format` - Format code with Prettier

## 🔐 GitHub Integration

This repository is connected to Lovable. All changes made in Lovable will be automatically synced to GitHub via commits on the main branch.

### Important Notes:
- ✅ Always pull before starting work
- ✅ Review changes before committing
- ✅ Keep the main branch stable
- ❌ Don't force push or rebase after connecting to Lovable
- ❌ Avoid merge conflicts by syncing frequently

## 📚 Stack

- **Framework**: TanStack Start (Vite)
- **UI**: React 19 + TypeScript
- **Styling**: Tailwind CSS 4
- **Components**: shadcn/ui + Radix UI
- **Forms**: React Hook Form + Zod
- **State**: React Query (TanStack Query)
- **Database**: Supabase
- **Package Manager**: Bun/npm

## 🚀 Deployment

This project is configured for deployment on:
- Netlify (via netlify.toml)
- Vercel
- Cloudflare Pages (Wrangler configured)
